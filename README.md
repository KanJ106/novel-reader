# Novel Overlay

一个 Windows 桌面小说悬浮阅读小工具。

## 功能

- 启动后可导入本地 `.txt` 小说
- 主界面只有小说文字，无边框、置顶显示
- 默认窗口更小，适合放在工位屏幕角落
- 按住鼠标左键可拖拽位置
- 鼠标滚轮直接按页前后切换，不再是上下滚动到模糊位置
- 点击一次小说窗口后进入阅读态，之后鼠标停在别处也能继续滚轮翻页
- 阅读态下只要在别处点击，小说会立刻自动隐藏
- 应用聚焦时按 `Space` 立即隐藏
- 按 `Ctrl + Alt + Space` 全局恢复显示
- 自动记住上次阅读进度、窗口位置、字体大小
- 内置书架，可重新打开最近读过的小说
- 自动识别常见章节标题，可从目录选择后跳转

## 操作

- `Ctrl + O`：导入新小说
- `Ctrl + B`：打开书架
- `Ctrl + T`：打开章节目录
- `Ctrl + -`：缩小字体
- `Ctrl + =`：放大字体
- `Left / Right`：上一页 / 下一页
- `Ctrl + Alt + Space`：恢复显示
- `Esc`：退出应用
- 右键文字区域：打开快捷菜单

## 阅读态说明

- 用 `Ctrl + Alt + Space` 唤出小说窗口
- 点击一次小说文字区域后，进入阅读态
- 阅读态下滚轮不依赖鼠标悬停位置，停在别处也能直接前后翻页
- 一旦在其它窗口点击，小说会马上隐藏

## 运行

安装依赖：

```powershell
pip install -r requirements.txt
```

启动应用：

```powershell
python app.py
```

也可以直接双击：

`run_novel_reader.bat`

## 说明

- 图形界面使用 Python 自带 `tkinter`
- 全局快捷键和鼠标监听使用 `pynput`
- 当前默认尝试 `utf-8`、`utf-8-sig`、`gb18030`、`utf-16`、`big5` 编码
- 阅读记录和书架保存在同目录下的 `reader_state.json`，该文件不会上传到 GitHub
- `Alt + Space` 是 Windows 常见系统组合键，如果和别的软件冲突，可以后续再改
